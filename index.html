Instrucciones para Utilizar PHP CON HTML

  1. Paso 1: Instala XAMPP
    Descarga XAMPP desde el sitio oficial: https://www.apachefriends.org/index.html.
    Instala XAMPP siguiendo las instrucciones del instalador. Elige la instalación completa con Apache, PHP y MySQL (u otros componentes si los necesitas).
  2. Paso 2: Inicia el servidor Apache en XAMPP
    Abre el Panel de Control de XAMPP.
    En el panel de control, localiza Apache y haz clic en el botón Start. Esto iniciará el servidor web que procesará los archivos PHP.
    Si Apache está en funcionamiento, verás un indicador verde y podrás acceder al servidor localmente a través de http://localhost.
  3. Paso 3: Crea un archivo PHP
    Ve a la carpeta donde XAMPP guarda los archivos para el servidor web. En la mayoría de las instalaciones, esta carpeta se llama htdocs y se encuentra en:
    Windows: C:\xampp\htdocs
    Crea una nueva carpeta dentro de htdocs para organizar tus archivos. Llamarla ejephp.
    Dentro de la carpeta ejephp, crea un archivo llamado index.php.
  4. Paso 4: Escribe código HTML y PHP en el archivo index.php
    Abre el archivo index.php con Visual Studio Code.
  5. Paso 5: Escribir en el HTML
    Copia y pega el siguiente código en index.php: 
    
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página PHP</title>
</head>
<body>
    <h1>¡Hola, mundo!</h1>

    <?php
        echo "<p>Este es un párrafo generado con PHP.</p>";
        $nombre = "Gaston";
        echo "<p>Mi nombre es $nombre.</p>";
    ?>
</body>
</html>
  6. Paso 6: Accede al archivo en el navegador
    Abre tu navegador y accede a la siguiente URL: http://localhost/ejephp/index.php.
